SMART BUILDING V1 - BUILD APK

Important:
This package is ready to be opened in Flutter, but the APK was not generated in this environment because Flutter SDK is not installed here.

Fast build steps on your computer:
1. Install Flutter.
2. Open terminal in this folder.
3. Run: flutter pub get
4. Run: flutter create .
5. Run: flutter build apk --release

Debug test APK:
flutter build apk --debug

The generated APK will be here:
build/app/outputs/flutter-apk/

Suggested filename to share:
smart-building-v1-test.apk
